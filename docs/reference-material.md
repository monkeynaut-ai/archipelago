
## Reference repos

### LangGraph and LangChain

#### [open-swe](https://github.com/langchain-ai/open-swe/tree/main)
No longer maintained, but provides examples of an agentic system for software development.  Gives examples of
- instructions for agents used in software engineering
- hooks
- tool use
- Deep Agents and SubAgent

This repo implements two  apps
- [SWE](https://github.com/langchain-ai/open-swe/tree/main) is a more mature and capable agentic system
- [SWE v2](https://github.com/langchain-ai/open-swe/tree/main/apps/open-swe-v2) uses LangChain's [Deep Agents](https://docs.langchain.com/oss/python/deepagents/overview)
